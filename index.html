Continuo motivado a aprender, evoluir e explorar novos horizontes, sempre guiado pela minha paixão por tecnologia e ciência de dados. Este portfólio é mais do que uma vitrine de projetos; é o reflexo do meu entusiasmo em transformar dados complexos em análises claras e acessíveis, capazes de gerar insights compreensíveis para qualquer público e não apenas para nós que estamos acostumados ao mundo dos datasets.

Projetos Ciência de Dados

Aplicativo Web: Previsão de Preços (Machine Learning) e Análise de Dados Automotivos.

Aplicativo web desenvolvido para oferecer uma análise completa e interativa do mercado de veículos (utilizando dados fornecidos por uma plataforma de vendas de carros), integrando técnicas de machine learning e visualização de dados. Com foco na experiência do usuário, o projeto permite explorar um conjunto de dados automotivos detalhado, investigando padrões e tendências importantes. Entre as principais funcionalidades estão filtros personalizados que possibilitam refinar a análise por marca, modelo, ano de fabricação e faixa de preço, além de gráficos dinâmicos que ilustram distribuições de preços, quilometragem e outras características dos veículos. Uma das grandes inovações é a integração de um modelo de machine learning, permitindo que o usuário insira as características do seu carro para a I.A. realizar predições de preços com base no histórico de dados, auxiliando tanto compradores quanto vendedores na tomada de decisões estratégicas. Este projeto combina ciência de dados e desenvolvimento web, resultando em uma ferramenta poderosa e prática para o mercado automotivo.

Como Utilizar: Clique na imagem ou no título acima, espere alguns segundos e você será redirecionado para o link do projeto. Todos os textos estão em inglês, e, se você preferir, clique com o botão direito do mouse e traduza a página.

Ferramentas Utilizadas: Streamlit (Aplicação Web), Scikit-Learn (Treinamento de I.A.), Pandas (Manipulação de Dados), CSS (Estilização), Plotly e Matplotlib (Visualização Gráfica), Scipy e Statsmodels (Testes de Hipóteses).

About Me

Bruno Caraffa

Since a little kid I've been extremely curious. The first dataset I ever had contact with was a world map that my dad gave me when I was 8. I memorized as many countries and the respective capitals and flags I could just for fun.

Data Science was what I needed to feed my constant curiosity in the adult life. During the pandemic I had the time to dive deeper into this world. Since then I've been working in projects envolving image classification, price prediction, sentiment analysis and clustering among others. I love to write Medium articles about those projects, and you can check them here.

I work with data since 2017 when I began my career in a auto retailing group. The dynamic pace of the retailing sector was the perfect environment to learn how data can be a leverage to companies. Graduated in Industrial Engineering in 2019 at the University of Brasilia and currently work as a Data Intelligence Coordinator at Wiz Co., where I lead a team of analysts to help the top management to evaluate the performance of business units with reports and make predictions for client churn and ensurance renewal using Machine Learning Pipelines.

Data Science Projects

Apache Spark MLlib vs Scikit-learn: Building Machine Learning Pipelines

Machine Learning pipelines simplify the coding process during a model implementation, making it cleaner and faster by improving the code readability with a logic workflow of the transformations during the model training. Pipelines are vital for ML projects involving large sets of data and with multiple data science team members involved. In this article, I compared pipelines from two of the main ML libraries: Scikit-learn and MLlib. Code implementation is presented to transform a raw dataset containing AirBNB listings and generate a prediction to the rent prices using a Random Forest Regressor. A mean average error of 67 euros was achieved through the pipeline model.

Behind the Scenes of a Deep Learning Neural Network for Image Classification

A dive into the math behind the neural networks with a code implementation of a deep neural network using only NumPy. Cat images were mixed with random images to train the model that was able to detect cats in pictures with 77% accuracy. The gradient descent algorithm was used to optimize the predictions and all of its steps were applied in the code using vectorized operations.

3-D Clustering of NBA Players with Plotly and K-Means

A clustering of over 4.000 former and current NBA (National Basketball Association) players by using the K-Means algorithm to group the players according to their career statistics. An interactive 3-D scatter plot was created with Plotly to enable cluster visualization. Later, a detailed cluster analysis is conducted to investigate the characteristics of each cluster and add explainability to the K-means model.

How I Used Web Scraping and Machine Learning to Solve my own Rental Problem

Is the price paid for rent fair? Are there any other rent options within the desired size and price ranges? In this article, I answed those questions for my real-life housing problem developing a data science application. I've created a code to web scrape over the listings and get the apartments data to train a price prediction model using the Random Forest Regressor. The model had a mean absolute error of under $140 predicting the prices of testing data. Later, using the Nearest Neighbors algorithm a recommendation system was created to return listing options based on the user's input of the apartment size, location, and amount of rooms. Finally, the application was hosted on Heroku using Streamlit for the front end and Docker to set up the application's environment.

Twitter Sentiment Analysis Using LSTM

A guide on how to apply the power of LSTM (Long Short Term Memory) in Recurrent Neural Networks to sentiment analysis with Twitter data. Python library SpaCy Natural Language Processing (NLP) techniques were applied in the pre-processing of tweets, such as lemmatization, stop word removal, tokenization, and vectorization. Later, using Keras I created a Sequential Model with four layers: an embedding layer, a spatial dropout, the LSTM layer itself, and a densely-connected layer with a sigmoid activation to generate the outputs. The model predicted whether the 2.104 evaluation tweets were positive or negative with over 87% accuracy and an F1 score of 0,878.
